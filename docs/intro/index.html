Introduction

Mercury is a cross-platform LAN clipboard and file-sharing app. It runs in your system tray, discovers other Mercury instances on the same network, and delivers copied text, images, and files between your machines.

Everything is encrypted with a shared passphrase you choose. There is no cloud, no account, and no sync history.

What Mercury does​

  • Clipboard sync: copy on one device, paste on another
  • File transfer: offer files to peers; they accept or decline
  • Tray-native UX: stays running in the background; open the window when needed
  • Local encryption: AES-256-GCM with PBKDF2 key derivation

What Mercury is not​

Clipboard managerNo history, search, or pinning. Mercury forwards live copies only.
Cloud serviceData stays on your LAN. Nothing is uploaded to a remote server.
Internet syncMachines must be on the same local network (or reachable LAN).

Requirements at a glance​

  1. Two or more computers on the same network
  2. The same passphrase on each machine
  3. Mercury running on each machine (tray icon visible)

When peers appear on the Home screen, sync is ready. See Getting Started for install steps and download links.

Logo​

The caduceus mark (winged staff, two snakes) refers to Mercury/Hermes, the messenger of the gods, not the medical Rod of Asclepius (one snake, no wings).
